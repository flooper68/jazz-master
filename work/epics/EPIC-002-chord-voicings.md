# EPIC-002 — Chord voicings

## Goal

A guitarist can browse, learn, and drill the essential jazz voicing vocabulary: shell voicings, drop 2, drop 3, across string sets and positions.

## Scope

- Voicing data model: chord quality × voicing type × string set × inversion → concrete grip
- Voicing generator or curated library for the core qualities (maj7, m7, 7, m7b5, dim7)
- Browse UI: pick a chord, see its voicings on chord diagrams
- Drill mode: prompt a chord + voicing type, reveal the grip, self-graded (got it / missed it)
- Track which voicings are learned vs. shaky

## Out of scope

- Progressions (EPIC-003), audio playback

## Depends on

- EPIC-001 (theory core, chord diagram component)
