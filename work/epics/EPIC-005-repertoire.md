# EPIC-005 — Repertoire

## Goal

Help guitarists build and maintain a book of jazz standards: track which tunes they know, at what depth, and what to review next.

## Scope

- Tune list: user adds standards they're working on (title, composer, form, default key, tempo range)
- Learning stages per tune: melody → chords → comping → soloing → memorized
- Chord-chart view for a tune's changes (user-entered; no copyrighted content bundled)
- Spaced review suggestions: which tunes are getting rusty
- Integration hooks: "practice this tune" links into progression/voicing drills for its changes

## Out of scope

- Bundled copyrighted lead sheets; audio playback of tunes

## Depends on

- EPIC-001 (theory core, persistence); richer integration needs EPIC-002/003
