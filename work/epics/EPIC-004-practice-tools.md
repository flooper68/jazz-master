# EPIC-004 — Practice tools

## Goal

The infrastructure of a practice session: a good swing-aware metronome, practice timers, and a session builder that strings drills into a structured routine.

## Scope

- Web Audio metronome: precise scheduling, tempo 40–300, subdivisions, swing feel, accent patterns (2 & 4)
- Practice timer / interval blocks ("10 min voicings, 10 min ii–V–I, 10 min repertoire")
- Session builder: compose blocks from other modules' drills into a saved routine
- Session log: what was practiced, for how long, at what tempo (feeds progress tracking)

## Depends on

- EPIC-001 (app shell, persistence)

## Done when

A user can build a 30-minute routine, run it with the metronome at 2 & 4, and see the session in their practice log.
