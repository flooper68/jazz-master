# Validation

## Artifact correctness

Use the approved drawing and inspection plan to check critical dimensions,
material/finish, and visible defects. Record the result against the batch and
sample. Exact methods and tolerances belong to the concrete production problem.

## Outcome effectiveness

Install a sampled conforming bracket in the target assembly and run the required
load trial. Record rework, fit, installation friction, deformation/failure, and
field feedback. A conforming part can still fail this outcome check.
