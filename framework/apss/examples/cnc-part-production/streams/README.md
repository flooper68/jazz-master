# Streams

This example consumes three heterogeneous streams declared in `SYSTEM.md`:
released CAD/drawings, machine/inspection evidence, and assembly/field feedback.
Native records may live elsewhere; the system preserves a usable reference or
topic summary in its work log.

The assembly-feedback grill asks for the attempted use, observed fit/load,
frequency, impact, and workaround. It does not force machine telemetry or CAD
revisions into the same conversational format.
